# Alla One Metronome 🥁

**Alla One** is a professional-grade, mobile-optimized metronome designed specifically for drummers. The name comes from the Hebrew slang "Alla One," referring to hitting the first beat of a measure with perfect precision.

Built as a **Progressive Web App (PWA)**, it offers a seamless experience on both mobile and desktop, focusing on high precision, ease of use, and practice tracking.

## ✨ Key Features

- **Radial Jog Wheel:** A natural, touch-responsive dial for smooth BPM adjustments.
- **High-Precision Audio:** Uses the Web Audio API for rock-solid timing, even at high tempos.
- **Tap Tempo:** Quickly find your BPM by tapping along to any rhythm.
- **Subdivision Support:** Practice with 1/4, 1/8, Triplets, and 1/16 notes.
- **Integrated Practice Tracker:** - Save custom exercises with their specific BPM.
  - Track your practice duration for each rudiment or exercise.
  - Reset timers and update BPMs with a single click.
- **Mobile First:** - **PWA Ready:** Install it on your home screen for a full-screen app experience.
  - **Wake Lock:** Prevents your phone screen from dimming or locking while you play.
  - **Responsive Design:** Adapts perfectly to any screen size.

## 🚀 How to Install

1. Open the [Live Demo Link](https://your-username.github.io/alla-one/) on your mobile browser.
2. **On iPhone (Safari):** Tap the **Share** button (square with arrow) and select **"Add to Home Screen"**.
3. **On Android (Chrome):** Tap the **three dots** and select **"Install App"** or **"Add to Home Screen"**.
4. Launch **Alla One** from your home screen and start practicing!

## 🛠️ Built With

* **HTML5 / CSS3** - Responsive UI and PWA Meta tags.
* **JavaScript (Vanilla)** - High-precision Web Audio API and Radial math.
* **Wake Lock API** - To keep the screen active during sessions.
