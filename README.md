# Alla One - Pro Metronome 🥁

**Alla One** is a professional-grade, mobile-first metronome designed specifically for drummers. The name is a nod to the musical slang "Alla One," representing the precision of hitting the first beat of a measure perfectly.

Built as a **Progressive Web App (PWA)**, it offers a rock-solid practice tool that lives on your home screen.

### 🚀 [Launch Alla One App](https://shayzinger738.github.io/alla-one/)

---

## ✨ Key Features

- **Mobile-Optimized UI:** A sticky header keeps the BPM display and visual beats visible even while scrolling through your practice list.
- **High-Precision Audio:** Uses the Web Audio API for stable, professional timing across all tempos.
- **Smart Subdivisions:** Support for 1/4, 1/8, Triplets, and 1/16 notes with distinct accents for better rhythmic clarity.
- **Dynamic Practice Tracker:** - Save custom exercises with unique names.
  - One-tap "Save BPM" to update exercises on the fly.
  - Independent practice timers to track your session duration.
- **Tap Tempo:** Quickly find the right speed by tapping along to any beat.
- **Professional Sound Profiles:** Toggle between "Wood" (classic) and "Sharp" (electronic) sounds to cut through your drum volume.

## 📱 Mobile Experience (PWA)

- **Fullscreen Mode:** Opens without browser bars when added to the home screen.
- **Screen Wake Lock:** Prevents your phone from dimming or locking during long practice sessions.
- **Responsive Design:** Carefully crafted for one-handed operation behind the drum kit.

## 📥 How to Install on Your Phone

1. Visit [Alla One](https://shayzinger738.github.io/alla-one/) in your mobile browser.
2. **On iPhone (Safari):** Tap the **Share** button and select **"Add to Home Screen"**.
3. **On Android (Chrome):** Tap the **three dots** and select **"Install App"** or **"Add to Home Screen"**.
4. Look for the **Vinyl Icon** on your home screen and start playing!

## 🛠️ Technical Overview

* **Engine:** Vanilla JavaScript / Web Audio API.
* **Storage:** LocalStorage (your exercise list stays on your device).
* **Styling:** CSS3 Flexbox/Grid with mobile-safe-area support.
